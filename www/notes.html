Notes

1. Not you, not your fault, but real

State plainly: not an accusation, not their sin

Name it: sihr — done to them by another's choice

Precedent: happened even to the Prophet ﷺ — no reflection on their standing

Reframe ask: not guilt, but cooperation — their will has to work alongside the ruqyah

2. The acquired state

Started as intrusion, but time + no removal = it embeds

Analogy: a path worn into ground — groove forms even without full consent

Name the resistance directly: if they feel reluctance despite wanting freedom, that's the groove, not insincerity

Reassure: struggle ≠ failure — this is a recognized stage, not backsliding

3. The corruption pattern (dose to the host — shorten/soften if scrupulous rather than resistant)

Never starts obviously evil — always small, plausible, sometimes even "good"-seeming

That's deliberate — an obvious evil gets rejected on sight

Small foothold → escalation, left unchecked

End point if unaddressed: shirk — misattributing power/reliance away from Allah

Purpose of naming this: recognition = permanent closure, not fear for its own sake

4. State during the session

Wudu if possible

Don't suppress reactions — let them happen, it's information not shame

If urge to stop arises: silently note "this is not me," continue anyway

Intention fixed: cure from Allah alone — not from you, not from the ruqyah itself as an object

5. State afterward

Daily practice matters, but intention behind it matters equally

At each pull-moment: recognize it, resist it — don't wait to feel ready first

Acting against the pull while still feeling it is the sabr, not a lesser version of it

Closing anchor: it was done by another's choice, holds power only by Allah's permission — freedom was never in doubt from Allah's side, only requires their steady return
